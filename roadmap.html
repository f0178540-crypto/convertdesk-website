Planned path to public release.

This roadmap tracks the work needed before ConvertDesk becomes a complete public Windows product with installer, domain, checkout and licensing.

Completed preparation

Temporary website

GitHub Pages website is online and available for review.

Core product pages

Home, Download, Pricing and Support pages are prepared.

Legal pages

Privacy Policy, Terms of Use and Refund Policy draft pages are prepared.

Release documentation

Release checklist, payment processor preparation and domain/payment setup plan are prepared.

Next required steps

1. Buy official domain

Preferred domain: convertdesk.com. Backup options: getconvertdesk.com, convertdesk.app, convertdesk.net.

2. Connect DNS to GitHub Pages

Configure GitHub Pages custom domain, DNS records and HTTPS.

3. Create official support email

Final planned support email: support@convertdesk.com.

4. Finalize Windows installer

Prepare ConvertDesk-Setup-1.0.0.exe after full test approval.

5. Set up payment processor

Recommended first attempt: Paddle. Backups: FastSpring and PayPro Global.

6. Add license activation

ConvertDesk Pro should not be sold publicly before activation and licensing behavior are defined.
