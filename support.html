What to include in a support request

To make support faster, include the technical details needed to reproduce the issue.

Windows version

Example: Windows 10 64-bit or Windows 11 64-bit.

ConvertDesk version

Example: ConvertDesk 1.0.0.

File type

Example: PDF, DOCX, PNG, MP4, ZIP, CSV.

Conversion target

Example: PNG to JPG, PDF to TXT, MP4 to MP3.

Bug report template

1. What happened?

Describe the problem clearly. Include any visible error message.

2. What did you expect?

Explain what you expected ConvertDesk to do.

3. Which file type did you use?

Include the input format and selected output format.

4. Can the issue be repeated?

Explain the steps needed to reproduce the same problem.

5. System information

Include your Windows version and ConvertDesk version.
